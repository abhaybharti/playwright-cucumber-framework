Key Folder/Files:

- config/ – Environment and browser config (ConfigManager.ts)
- pages/ – Page Object classes (like HomePage.ts)
- support/hooks.ts – Test lifecycle hooks
- support/PageFixture.ts – Shared access to page, context, browser
- support/locators/ – Centralized locators for UI
- tests/features/ – Gherkin BDD feature files
- tests/steps/ – Step definitions mapped to each page
